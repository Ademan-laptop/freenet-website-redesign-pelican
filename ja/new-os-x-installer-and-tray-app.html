New OS X installer and tray app

Sat 12 December 2015
By Freenet Project Inc.

The Freenet installer for OS X is now a native application. It should fix several major installation problems OS X users have encountered in the past.

It also includes an updated tray application with bugfixes and new features.

All Freenet users on OS X are encouraged to download the new tray application.
